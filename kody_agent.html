The bottleneck
~3 hrs
Average engineering time lost daily to context-switching between triage, investigation, and implementation.

Well-understood bugs sit in backlog not because they're hard, but because no one has bandwidth to switch context into them right now.

Context-switch cost is high
It takes ~23 minutes to fully regain focus after an interruption. Triage tickets create constant interruptions.
Issue-to-PR latency is long
Well-scoped bugs can still take hours or days to ship because writing the code requires a dedicated block of focus time.
Most triage work is execution, not judgment
Reading the codebase, finding the pattern, writing the fix, running tests. This is exactly what an AI agent is optimized for.

Safety model

Six layers between kody and production.

Every kody PR is sandboxed and ephemeral. The PR review is the final gate, but we don't rely on that alone.

"The walls matter more than the model."
Deterministic constraints (tests, linting, tool allowlists, required human review) are what make autonomous code generation safe to deploy. Model quality is secondary to the structure around it.
Stripe Blueprint architecture
01
Classification Gate
Fast Claude API call before the main run. Filters true no-ops (vague, non-actionable requests) before spending compute on a full session.
02
Tool Allowlist (Agent Mode)
Explicit --allowedTools in agent mode. Hard blocks on rm, curl, wget, and db commands. Cannot be overridden by the prompt.
03
Deterministic Validation
Tests and linting run as separate steps with real exit codes. kody never self-validates. If StandardRB fails, the run fails, not "Claude says it looks fine."
04
Bounded Retry Loop
Maximum 2 CI rounds. After 2 failures, kody opens the PR anyway with failure context attached. It never spins indefinitely.
05
Existing CI Pipeline
Every kody PR runs Brakeman, StandardRB, ERB lint, and the full RSpec suite, the same gates as any human-authored PR.
06
Mandatory Human Review
PRs labeled kody never auto-merge. Branch protection requires human approval. Structural, not configurable.
