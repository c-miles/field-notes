# Field Notes

A place to park HTML visual explainers for PRs, design reviews, and anything that's easier to show than to write out.

Each file is a standalone page. Commit, push, and GitHub Pages serves it at `https://c-miles.github.io/visual-explainers/<filename>.html`. The root index auto-lists every HTML file in the repo, so new additions show up on their own.

Most of what's in here comes from the visual-explainer skill. Anything self-contained works though. Diagrams, walkthroughs, interactive demos, the odd slide deck.
